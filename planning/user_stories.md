# User Stories

Reference the Writing User Stories final project guide in the course portal for more information about how to complete each of the sections below.

## Outline User Roles

***Tourist:*** A user who is new to a city (doesn't know their way around) and is looking for a nearby bathroom.

***Registered User:*** A user who wants to be able to favorite, and rate bathrooms.

## Draft User Stories

1. As a user, I want to find nearby public restrooms based on my location, so that I can quickly access the closest option when needed.

2. As a tourist, I want to find public restrooms in an unfamiliar area, so that I can navigate the city more comfortably.

3. As a user, I want to view details about a restroom (rating, description, images), so that I can decide if it meets my standards.

4. As a user, I want to submit a rating for a restroom, so that I can share my experience with others.

5. As a user, I want to upload images of restroom conditions, so that others can visually assess cleanliness and safety.

6. As a tourist, I want to see top-ranked restrooms nearby, so that I can quickly choose the best available option.

7. As a user, I want to filter restrooms by criteria (e.g., rating, distance, accessibility), so that I can find options that match my preferences.

8. As a registered user, I want to create an account and log in, so that I can manage and track my reviews.

9. As a user, I want to save or bookmark restrooms, so that I can quickly access them later.

10. As a user, I want to edit or delete my reviews, so that I can update or correct my feedback.
